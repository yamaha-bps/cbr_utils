◆ apply

template<typename... Args>
template<template< typename... > typename T, typename... Ts>
using cbr::TypePack< Args >::apply = TypePack<T<Args, Ts...>...>

Application of a given templated type over each element of the pack.

Result is another TypePack.

For example:

template<class T>
struct S{};
using T = TypePack<int,double,float>;
using T2 = T::apply<S,char,void*>;

Then T2 == TypePack<S<int,char,void*>,S<double,char,void*>,S<float,char,void*>>.

Template Parameters
TTemplated type to use for application.
TsOptional types to append to T's template parameters.

◆ member_t

template<typename... Args>
template<template< typename... > typename T, typename... Ts>
using cbr::TypePack< Args >::member_t = TypePack<typename T<Args, Ts...>::type...>

Apply a given type over pack and extract the "type" typedef out of each element of the resulting pack.

Result is another TypePack.

Example:

template<class T1, class T2>
struct S{using type = std::make_unsigned_t<T1>;};
using T = TypePack<int,unsigned int>;
using T2 = T::member_t<S,char>;

Then T2 == TypePack<unsigned int,unsigned int>.

template<typename... Args>

using cbr::TypePack< Args >::member_v = typename detail::member_v_impl<T, Args...>::type

Apply a given type over pack and extract the "value" static member variable out of each element of the resulting pack.

Result is an std::integer_sequence.

Example:

template<class T>
struct S{static constexpr std::size_t value = sizeof(T);};
using T = TypePack<int,double,char>;
using T2 = T::member_v<S>;

Then T2 == std::integer_sequence<std::size_t,4,8,1>.

TTemplated type to use for application.
