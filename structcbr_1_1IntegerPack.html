Detailed Description

template<typename Type, Type... Vals>
struct cbr::IntegerPack< Type, Vals >

Compile time pack of integers.

Similar to std::integer_sequence but with more functionalities.

Template Parameters
TypeAn integer type to use for the elements of the pack.
ValsA non-type parameter pack representing the pack.

using cbr::IntegerPack< Type, Vals >::append = typename detail::_integerpack_cat<this_t, IntegerPack<Type, Vs...> >::type

Append values to the pack.

Result is another IntegerPack.

Example:

using T = IntegerPack<int, 0, 1, 2>;
using T2 = T::append<3, 4>;

Then T2 == IntegerPack<int,0,1,2,3,4>.

using cbr::IntegerPack< Type, Vals >::apply = TypePack<T<Vals, Vs...>...>

Application of a given templated type over each element of the pack.

Result is a typepack.

For example:

template<int i, int j, int k>
struct S{};
using T = IntegerPack<int, 0, 1, 2>;
using T2 = T::apply<S,9,8>;

Then T2 == TypePack<S<0,9,8>,S<1,9,8>,S<2,9,8>>.
