inlineconstexpr

Always false constexpr constant.

Usefull to allow compilation of:

template<class T>
void foo()
{
if constexpr (std::is_signed_v<T>) {
// doStuff...
} else {
static_assert(false_v<T>,"Unsupported type");
}
}

◆ is_specialization_v

template<typename T , template< typename... > class Ref>
constexpr bool cbr::is_specialization_v = is_specialization<T, Ref>::value
inlineconstexpr

Check if type is a specialisation of another type.

For example:

constexpr auto v1 = is_specialization_v<std::vector<double>,std::vector>;
constexpr auto v2 = is_specialization_v<std::vector<double>,std::tuple>;

Then v1 == true but v2 == false.

Template Parameters
TType to check if specialization of Ref or not.
RefReference type to check if T is a specialization of or not.
