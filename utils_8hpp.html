const bool  fullPrecision = false 
)
inline

Convert timepoint into string.

String format is Y-m-d_H-M-S(.nanoseconds). See https://en.cppreference.com/w/cpp/chrono/c/strftime. Example:

const auto now = std::chrono::system_clock::now();
const auto s1 = dateStr(now);
const auto s2 = dateStr(now,true);

Then s1 == "2022-01-01_15-13-54", and s2 == "2022-01-01_15-13-54.142542392".

Template Parameters
clock_tType of the clock. Defines the type of the input arguments.
Parameters
currentTimeTimepoint to convert into a string.
fullPrecisionIf true, adds nanosecond value to output string (default: false).
Returns
std::string representation of the input timepoint.
