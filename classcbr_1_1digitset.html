Detailed Description

template<std::size_t N_DIGITS, std::size_t BASE = 10>
class cbr::digitset< N_DIGITS, BASE >

Extention of std::bitset for bases > 2.

Allows to easily extract the digits of a number in a given base.

Template Parameters
N_DIGITSMaximum number of digits to use for representation.
BASEBase used to represent the number (default is 10).
