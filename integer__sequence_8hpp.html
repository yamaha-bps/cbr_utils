◆ iseq_apply_t

template<typename ISeq , template< typename ISeq::value_type... > typename T>
using cbr::iseq_apply_t = typedef typename iseq_apply<ISeq, T>::type

Application of a given templated type over each element of the sequence.

For example:

template<int i, int j, int k>
struct S{};
using T = std::integer_sequence<int, 0, 1, 2>;
using T2 = iseq_apply_t<T,S>;

Then T2 == S<0,1,2>.

Template Parameters
ISeqInteger sequence to apply templated type to.
TTemplated type to use for application.

Create an std::index_sequence.

  • If Vs is 2 elements, create an index sequence of all values between this 2 elements (2 elements included).
  • If Vs is 1 element, create an index sequence from 0 to Vs (Vs excluded).
